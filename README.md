# Assistente Financeiro Pessoal

Projeto de console em linguagem C desenvolvido com foco em aprendizado de programação durante o curso de ADS.

---

## 🎯 Objetivo

Aplicar conceitos fundamentais de programação estruturada na construção de um sistema de controle financeiro pessoal.

---

## 🚀 Evolução do Projeto

### Versão 1
- Entrada de dados (salário, gastos, dívidas)
- Cálculo de saldo simples

### Versão 2
- Validação de entradas
- Estruturas condicionais

### Versão 3
- Estruturas de repetição
- Simulação de quitação de dívidas

### Versão 4
- Menu interativo com switch/case
- Organização por funcionalidades

---

## 🛠️ Tecnologias

- Linguagem: C (C99/C11)
- Compilador: GCC (MinGW)
- IDE: VS Code
- Controle de versão: Git/GitHub

---

## 🔮 Próximos passos

- Arrays
- Funções modularizadas em arquivos
- Structs
- Persistência em arquivos

---

## ▶️ Como executar

1. Clonar o repositório
2. Compilar com GCC
3. Executar o arquivo principal
